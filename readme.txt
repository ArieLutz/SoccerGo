----------------------------  App SoccerGo  --------------------------------

* Tecnologías utilizadas

    React native
    Expo
    React navigation


(NECESARIO) Guia de Instalación de componentes 

USAR CMD para la instalacion de cada una.
* Librerias necesarias a instalar para APP SoccerGo

1 Expo ( necesario para el entorno de desarrollo).

	fuera de la carpeta en donde se encuentra el proyecto:
	Escribir el siguiente comando:
	
	- Expo init/ seleccionar la carpeta del proyecto (ya existente) y la opción (BLANK "la primera").*


2 React Navigation.

- En la carpeta del proyecto escribe los comandos:
	npm install @react-navigation/native
	expo install react-native-gesture-handler react-native-reanimated react-native-screens react-native-safe-area-context @react-native-community/masked- view
	
	npm install @react-native-community/masked-view

3 Navegación en pilas:
	npm install @react-navigation/stack
	npm install react-native-screens



4 React Native Elements (Componentes para estilizar)
 	npm install react-native-elements

5 Axios (componente para las dependencias)
	npm i axios

6 Para trabajar con las fechas
	npm install date-fns

7 Verificacion de email en los input
	expo install email-validator
